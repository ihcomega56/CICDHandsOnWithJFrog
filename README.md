# CICDHandsOnWithJFrog

大まかなシナリオ

- Javaアプリケーションをビルドする: エラーがあったら取り除いてビルド成功させる
- Artifactoryにアーティファクトを保管する
- セキュリティスキャンを行う: エラーとして検知するレベル以上の脆弱性が見つかったら修正する
- プロモートする
